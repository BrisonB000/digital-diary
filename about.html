About IRON DIARY

IRON DIARY is a smart and secure digital diary web application designed to help users store their daily memories, thoughts, and experiences in one place. Inspired by the power and elegance of Iron Man, this diary uses a red, gold, black, and arc-reactor-blue theme to create an attractive and futuristic user experience.

Project Purpose

The main purpose of IRON DIARY is to replace traditional paper diaries with a modern web-based diary system. Users can register, log in securely, and write diary entries for each day. The application is designed so that each date has a dedicated diary page, making it easy to manage daily records in an organized way.

Key Features

  • Secure user registration and login using Supabase authentication
  • One diary entry for each date with the ability to update it anytime during the day
  • Calendar-based diary viewing for quick access to daily memories
  • Edit and save diary content whenever needed
  • Modern Iron Man inspired interface with glowing UI elements
  • Cloud-based storage for reliable and easy access

Technologies Used

  • HTML for webpage structure
  • CSS for styling and Iron Man themed design
  • JavaScript for interactivity and diary logic
  • Supabase for authentication and cloud database storage
  • Vercel for website deployment

Future Enhancements

In future versions, IRON DIARY can be enhanced with AI-powered title suggestions, mood tracking, reminder notifications, search options, and richer calendar visualization for a smarter diary experience.
